**Luna Calendar**
---
A desktop application written in C/C++ that displays a lunar calendar with a graphical interface. The app lets the user enter a timezone, then calculates and shows the relevant lunar phase for that place and time.


Build and Run:

1) Clone or download the repo

2) Open it in your C++ IDE or Qt Creator

3) Build the project

4) Run the executable

5) Enter the timezone in the app
